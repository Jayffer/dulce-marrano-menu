# Fuentes

Coloca aquí los archivos de la fuente **Pink Broccoli**:

- `PinkBroccoli-Regular.woff2`
- `PinkBroccoli-Regular.woff`
- `PinkBroccoli-Regular.ttf`

Si tienes otros formatos o variantes (bold, italic), puedes añadirlos también.

**Montserrat** se carga automáticamente desde Google Fonts, no necesitas descargarla.
